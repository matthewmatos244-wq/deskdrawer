# DeskDrawer MVP

DeskDrawer is an AI workspace memory app.

## Core user value
Users can connect Gmail and Google Drive, ingest their content, and ask natural-language questions across all synced data.

## MVP features
1. User authentication
2. Connect Gmail
3. Connect Google Drive
4. Background sync for emails and files
5. Parse and chunk synced content
6. Store embeddings and metadata
7. Chat UI for retrieval-based Q&A
8. Responses must show source citations
9. Search results should include:
   - title
   - source type
   - date
   - snippet
10. Simple dashboard showing sync status and recent indexed items

## Non-goals
- No Slack yet
- No browser recording yet
- No enterprise admin
- No mobile app
- No autonomous agents

## Tech preferences
- Next.js frontend
- API routes or Python service if needed
- Postgres
- Clean modern UI
- Strong typing
- Clear folder structure
